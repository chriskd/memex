# MCP Server Setup (Deprecated)

**Deprecated:** Memex no longer ships or supports an MCP server. Use the `mx` CLI instead.

If you previously configured an MCP server, remove the `memex` entry from your Claude Desktop or Claude Code config to avoid startup errors.
